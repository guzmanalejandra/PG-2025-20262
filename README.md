# Proyecto de Graduación 2025 — PG-2025-20262  
## Doritapp: Aplicación móvil para apoyo a la alfabetización inicial

## 1. Descripción General

Doritapp es una aplicación móvil orientada a apoyar el proceso de alfabetización inicial en adolescentes y adultos guatemaltecos. El proyecto integra principios pedagógicos como el principio alfabético, el método fonético y la andragogía digital, junto con un enfoque de diseño centrado en el usuario.

El objetivo principal es ofrecer una herramienta accesible, intuitiva y adaptada a las necesidades de personas con poca o nula experiencia lectora, facilitando el aprendizaje de letras, fonemas, sílabas y oraciones sencillas mediante actividades auditivas y visuales.

---

## 2. Objetivos del Proyecto

### Objetivo general  
Desarrollar un prototipo funcional de una aplicación móvil que apoye la alfabetización inicial de adultos jóvenes y mayores, integrando prácticas pedagógicas apropiadas y elementos de accesibilidad.

### Objetivos específicos  
- Implementar un conjunto de lecciones basadas en el principio alfabético y el método fonético.  
- Incorporar actividades prácticas con retroalimentación auditiva y visual.  
- Integrar opciones de accesibilidad dirigidas a adultos con baja alfabetización digital.  
- Validar el diseño y la usabilidad mediante pruebas con usuarios reales.

---

## 3. Tecnologías Utilizadas

- React Native (Expo)  
- TypeScript  
- Expo Speech (TTS)  
- AsyncStorage  
- Google Fonts (Outfit, Atkinson Hyperlegible)  
- Activos locales (imágenes y audios)  
- Diseño accesible y responsivo  

---


## 4. Ejecución del Proyecto

### 4.1 Requisitos Previos
- Node.js  
- Expo CLI  
- Dispositivo físico con Expo Go o emulador Android/iOS  

### 5.2 Instalación

Desde la carpeta `src`:

```bash
npm install
```

### 5.3 Ejecución

```bash
npm start
```
